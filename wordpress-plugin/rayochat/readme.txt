Integra RayoChat AI Customer Care nel tuo sito WordPress per fornire supporto clienti automatizzato e intelligente.

== Description ==

RayoChat è un plugin per WordPress che ti permette di integrare facilmente un widget di customer care basato su intelligenza artificiale nel tuo sito.

**Caratteristiche principali:**

* Widget chat elegante e moderno con design ispirato a Stripe
* Risposte automatiche basate sulla tua documentazione aziendale
* Sistema di valutazione delle conversazioni
* Completamente personalizzabile (colori, posizione)
* Design responsive per mobile e desktop
* Zero configurazione complessa

**Come funziona:**

1. Installa il plugin
2. Ottieni la tua API key dal pannello RayoChat
3. Inserisci l'API key nelle impostazioni del plugin
4. Il widget apparirà automaticamente sul tuo sito

== Installation ==

1. Carica la cartella `rayochat` nella directory `/wp-content/plugins/`
2. Attiva il plugin attraverso il menu 'Plugins' in WordPress
3. Vai in Impostazioni > RayoChat per configurare il plugin
4. Inserisci la tua API key e salva le impostazioni

== Frequently Asked Questions ==

= Dove ottengo la mia API key? =

Puoi ottenere la tua API key registrandoti su https://yourdomain.com e creando un account.

= Il widget è personalizzabile? =

Sì! Puoi personalizzare il colore primario e la posizione del widget dalle impostazioni del plugin.

= Il widget funziona su mobile? =

Sì, il widget è completamente responsive e funziona perfettamente su dispositivi mobile.
